### üìê Formatting Rules

Make sure to follow these formatting guidelines for all your documents:

#### **Fonts**

- Use **Montserrat** for *all* text, including titles, headers, body text, bullets, and captions. 
- Avoid using any other fonts like Cambria, Arial, or Times.

#### **Headers**

- **Title Block:**
    - Use Montserrat, bold, blue, and a larger size (approx 16 pt).
    - Place the subtitle directly under the title in blue Montserrat (approx 14 pt).
  
- **Heading 1:**
    - Use Montserrat, bold, blue, at a medium size (approx 13 pt).
  
- **Heading 2:**
    - Use Montserrat, bold, blue, at a slightly smaller size (approx 12 pt).

#### **Spacing**

- Set line spacing to **1.0**.
- Add a **6pt space** after each paragraph.
- Do not add extra spacing before paragraphs to keep sections tight and clean.

#### **Bullets**

- Use **circle bullets (‚Ä¢)**. 
- Avoid hyphens or dashes as bullets.
- Maintain indentation equal to Google Docs default settings.
